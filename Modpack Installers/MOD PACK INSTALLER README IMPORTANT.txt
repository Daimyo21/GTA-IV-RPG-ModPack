###IMPORTANT####
*This Installer will NOT overwrite your personal save files, optional overwriting of INI/Mod files is up to you during install

*NOTE:  If this is an updated modpack installer, please read changelogs and overwrite the files that have changed (it will ask you during install)

*THIS INSTALLER ONLY INCLUDES MODS THAT HAVE MODIFIED INIS AND MOD FILES, NOT THE INI FILES THAT HAVE NO MODS


Alternatively, you can install in a seperate directory and manually copy and paste files over

ENJOY!
